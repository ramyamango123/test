Negative Test Cases

  • When the search text entered in lowercase or uppercase, you shd get the same result
  • When you keep search text field empty and do search, it should not search for any products
  • When you try to enter the text beyond the text field limit, it should not accepted
  • When you enter some text that doesn’t match the database, a message as “your search did not match any products” should be displayed
  • When you copy and paste the search results URL in different window or browser, it should display the same set of results
  • When you enter the same search text in diff browsers, the results obtained should be exactly similar
