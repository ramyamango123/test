Course Description

An introduction to the open-source Selenium Integrated Development Environment, used for automating the testing of web pages. Topics covered will include: download and installation; user forums; comparison of tool with other similar products; record/replay; use of regular and globbing expressions within patterns; use of XPath expressions and the DOM to locate page elements; test suite execution via TestRunner; user extensions; the Selenium command set; and the use of JavaScript snippets as arguments to Selenium commands.

One hour lecture, one hour lab.
