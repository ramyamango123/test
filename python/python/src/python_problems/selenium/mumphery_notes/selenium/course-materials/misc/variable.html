Selenium Variables

  1. To store a value in a variable, call a Selenium function that has variable as one of its arguments.

  2. To specify this variable, simply give any name you wish. No special syntax (such as Perl requiring the preceding $) is required to store a value in a variable.

  3. To obtain the value of a variable (variable substitution), a special syntax is required: ${variablename}

  4. echo ${variablename} can be used to debug/test Selenium tests.

  5. echo is only supported starting with SIDE 1.0 beta.
