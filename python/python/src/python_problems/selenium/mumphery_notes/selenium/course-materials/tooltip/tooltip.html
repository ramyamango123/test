Tooltip Testing

  1. If the tooltip is just that provided by the browser via the title attribute of an img tag, then there's no way to record this.

  2. Instead, follow these steps:

    • Use verifyAttribute to verify the text in the title attribute.

    • Use an element locator, such as an xpath expression, as the first argument.

      One can use record to capture an appropriate locator instead of figuring it out by oneself. For example, click on a clickable image in order to capture a suitable locator, then change the command from click to verifyAttribute.

    • Add @title at the end of the first argument to further qualify the locator.

      NOTE: Do not leave any space between the captured locator and the added @title.

    • Use a pattern (glob:, regexp:, exact:) as the third argument.
