Test Automation Assignment

  1. Select a portion of the De Anza College web site as your SUT.

  2. Create at least 25 individual standalone test cases for your SUT. Each test case should contain at least one verify or assert command. Be liberal with your HTML comments which document each test case.

  3. Create a test suite which can be used to run all 25 test cases against Firefox.

  4. Write up a short description of the three most important or interesting points you learned while creating the test suite and test cases.
