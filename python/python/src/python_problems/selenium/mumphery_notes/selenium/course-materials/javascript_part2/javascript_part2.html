More JavaScript

  1. Using your own JavaScript functions with SIDE

    STEPS:

    1. Create a user-extensions.js file (can have different name) with whatever JavaScript functions/methods are desired.

    2. Select Options=>Options=>General from SIDE.

    3. Enter the path to the user-extensions.js file in the Selenium Core extensions field, then select the OK button to save the change.

    4. Close and re-open SIDE. (Critical step!)

    5. Create a test that calls the function. The function can be called by a SIDE command requiring a snippet or by one requiring an expression. The latter would require the javascript{} syntax as usual.

      • storeEval...function-call...variablename

      • store...javascript{function-call}...variablename

    LAB EXERCISE JAVASCRIPT_PART2-3

    Write a Selenium program that does the following:

    1. Creates a variable with the value of your name in title-case, e.g., Abraham Lincoln.

    2. Uses echo and a call to a JavaScript function from your user-extensions.js file to display your name in all upper-case in the log file.

    3. Uses echo and a call to a JavaScript function from your user-extensions.js file to display your name in all lower-case in the log file.

    4. Uses echo and a call to a JavaScript function from your user-extensions.js file to display your name enclosed in single quotes in the log file.

    5. Uses echo and a call to a JavaScript function from your user-extensions.js file to display your name enclosed in double quotes in the log file.

    6. Uses echo and a call to a JavaScript function from your user-extensions.js file to display your name in lastname, firstname format in the log file.

    For example, my solution to this program produced log output that looked like this... 

    [info] echo: EDITH WHARTON
.
.
.
[info] echo: edith wharton
.
.
.
[info] echo: 'Edith Wharton'
.
.
.
[info] echo: "Edith Wharton"
.
.
.
[info] echo: Wharton, Edith

    NOTE: Your program may contain only ONE occurrence of your name! You must store your name in a variable and manipulate the variable to produce the output above.

    HINT: You've already learned about JavaScript's concatenation operator and a String method which can convert a string to all uppercase. There's a similar method for converting a string to all lowercase. And there are also String methods named indexOf and substring which may prove helpful.
