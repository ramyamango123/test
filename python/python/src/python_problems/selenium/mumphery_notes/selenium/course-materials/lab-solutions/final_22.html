Midterm Take-Home Programming Problem #22

Create a single-file SIDE test case for the following test case specification. When you have finished testing it, email it to mamp@earthlink.net with a Subject: line of SELENIUM FINAL #22. You have until midnight tonight to mail it.

WARNING: Please be very careful to not accidentally submit the form with four valid values. We do not want a host of bogus comments landing in any campus email-boxes!

HINT: This problem is a test of your ability to synthesize what you've learned in class to research and utilize new (to you) Selenium functionality. In other words, you may need to skim through the reference docs, looking for a needed command or two.

REQUIRED: Store your valid values in variables, and access those variables for each of the four tests. Do NOT use a literal such as a value for the name field more than once.

Action Expected Results
  1. Point your browser to the De Anza College Questions and Comments Form.

  2. Enter values for email, comment, and recipient, but NOT name.

  3. Click the Submit button.

  4. Confirm the pop-up.
Verify that the landing page contains the string Please enter your name.
Return to the form.
  1. Enter values for name, comment, and recipient, but NOT email.

  2. Click the Submit button.

  3. Confirm the pop-up.
Verify that the landing page contains the string Correct e-mail address required.
Return to the form.
  1. Enter values for name, email, and recipient, but NOT comment.

  2. Click the Submit button.

  3. Confirm the pop-up.
Verify that the landing page contains the string No comments entered.
Return to the form.
  1. Enter values for name, email, and comment.

  2. Select the "-----------" recipient.

  3. Click the Submit button.

  4. Confirm the pop-up.
Verify that the landing page contains the string Error: recipient.
Return to the form.
