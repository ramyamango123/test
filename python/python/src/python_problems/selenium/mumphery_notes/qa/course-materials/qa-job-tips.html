QA Job Tips

  • Take a QA course. (It's unlikely any of your competitors will have done this, so a course will set you apart from them, plus it should impress the hiring manager with your sincerity.)

  • Learn technical skills relevant to the type of QA job you want. For example, a Web-page tester in a Linux environment would most likely need experience with HTML, JavaScript, CSS, Linux, PHP, and a tool to automate UI tests such as Selenium, WinRunner, QuickTest Pro, TestComplete, SilkTest, etc.

  • Gain real-world experience in the open-source world. For example, join the Mozilla QA Community.

  • Attend meetings of a professional organization, such as the SSQA.
