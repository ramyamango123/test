Testopia: Searching for Test Cases

  1. Select Search from Testopia portion of tool panel. (It's the fifth link.)

  2. Select whatever criteria you want or none at all if you want to see all the test cases.

  3. Select the Submit button on the search page.

  4. Select the link of an individual test case to see it expanded.
