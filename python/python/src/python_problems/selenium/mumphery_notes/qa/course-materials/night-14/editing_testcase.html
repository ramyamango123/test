Testopia: Editing Set Up, Break Down, Action, or Expected Results Fields

  1. Do a search for all test cases. Then select the ID of the test case to be edited.

  2. Select the Edit Document link to the right of the field to be edited.

  3. Wait for the toolbar to appear. Select the <h> icon.

  4. Add new text if none was there before, or edit the existing text.

  5. Select any of the many Commit buttons on the page.

  6. Receive confirmation in the form of a Test case updated message.
