  • Lab exercises:

    • Email yourself a hyperlink (most mail programs provide this functionality) of a Bugzilla query. Then log out of Bugzilla (assuming you were even logged in to start with). Finally, check whether the link for the query in the email works. You'll need to be able to send me clickable Bugzilla queries as part of your midterm.

    • A Saved Search cannot be emailed to another user. Convert one of your saved searches into a regular search, and email it to a fellow student. Ask that student to try the link to make sure it works.

      Steps for the conversion of a saved search into a regular search...

      1. Select the saved search of interest.
      2. Select the Edit Search link on the SERP (Search Results Page).
      3. Select the Search button on the Advanced Query page.
      4. Select the URL from the new SERP.
