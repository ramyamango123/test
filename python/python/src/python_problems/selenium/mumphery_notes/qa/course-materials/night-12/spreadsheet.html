HALF-BUY-TC-1 Check for misspellings, grammatical and punctuation mistakes, etc. No misspellings, grammatical, punctuation, formatting, etc. mistakes should appear in the Buy module.  
HALF-BUY-TC-2 Select the Advanced Search link to make sure it's working. The browser should display the Books Advanced Search page.  
HALF-BUY-TC-3 Negative: Enter a nonexistent ISBN, followed by selecting the Buy button. The browser should display a page explaining that no book with that ISBN number exists, and offering suggestions for remedying the problem. At present, this action leads to a half.com "page not found" page. Without specs, it's not clear whether this is a very poor user experience or an actual bug in that an error page should be the landing page.
