Testopia: Creating a New Run

  1. Select New Run from Testopia portion of tool panel.

  2. Enter a numeric plan id for an existing test plan.

  3. Select the Use This Test Plan button.

  4. Select the View All link in order to see all of the test cases at once.

  5. Select the Select All link or check as many individual test cases as are desired for inclusion in this run.

  6. Enter a value for the Summary field.

  7. Select a value for the Build field.

  8. Select a value for the Environment field.

  9. Select a value for the Product Version field. ("unspecified" is okay.)

  10. Select a value for the Run Manager field.

  11. Select the Submit button.

  12. Receive confirmation in the form of a pop-up that gives the id for the new test run.
