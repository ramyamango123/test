      • Bring up the individual test plan which has the test cases you wish to copy to another test plan.

      • Select the Test Cases tab.

      • Select the test cases you wish to copy.

      • Bring up the context menu and select the Copy or Link Selected Test Cases to Plan(s)... option.

      • Select the destination test plan(s).

      • Select the Create a copy link.

      • Select the Submit button.

      • View new test case id in confirmation pop-up.
