Testopia: Creating a New Build

  1. Select Product Dashboard from Testopia portion of tool panel.

  2. Select the appropriate Product from the leftmost frame of the dashboard.

  3. Select the Builds tab.

  4. Select the + icon (Add a new Build).

  5. Enter a value into the Name field. (The other two fields are optional.)
