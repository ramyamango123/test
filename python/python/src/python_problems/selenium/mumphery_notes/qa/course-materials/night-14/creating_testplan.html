Testopia: Creating a Test Plan

  1. Select New Plan from Testopia portion of tool panel.

  2. Enter a name for the test plan in the Name field.

  3. Select a product from the drop-down Product.

  4. Select the type of test plan from the drop-down for Plan Type.

  5. Select unspecified or other more appropriate value for the Product Version drop-down.

  6. Enter a Plan Document if desired.

  7. Select the Submit button.

  8. Receive confirmation in the form of a pop-up message. Note the numeric plan id, as this will be needed for creating test cases for this test plan.
