Testopia: Creating a Test Case

  1. Select New Case from Testopia portion of tool panel.

  2. Enter a value for plan id, then select Use This Test Plan.

  3. Enter a Summary value.

  4. Select CONFIRMED for Status field.

  5. Fill in the test case's action.

  6. Fill in the test case's expected results.

  7. Select a Priority and Category (default is okay.)

  8. Select the Submit button.

  9. See verification that new test case has been created--Test Case n: heading at top of page.
