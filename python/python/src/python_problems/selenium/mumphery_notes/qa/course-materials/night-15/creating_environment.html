Testopia: Creating a New Environment

  1. Select Product Dashboard from Testopia portion of tool panel.

  2. Select the Environments tab.

  3. Select the + icon (Add an Environment).

  4. Enter a value into the Name field of the pop-up.

  5. Select the product with which this environment should be affiliated from the dropdown menu for Product in the pop-up.

  6. Select the Create button on the pop-up.

  7. See confirmation Test environment <n> Created message in a new pop-up.
