Testopia: Adding Results to a Run

  1. Do a search for all test runs, and then select the id link for the one you wish to update.

  2. Select the View All link to see all the test cases in a scrolling list.

  3. Select the table row for an individual test (not the test case's id field).

  4. Select the Mark as PASSED or Mark as FAILED icon as appropriate for each test which has been executed, adding comments via the Notes tab and/or attaching bugs via the Bugs tab.

NOTES

  • Not all the results have to be entered at once! A few can be entered, and then this process can be continued again later. The only difference between the initial update and later ones will be that later sessions will already show status for those tests which have been updated.

  • Results can be changed if you discover you've made a mistake on a test's status. Simply return to the test run, expand the test case for which the mistake was made, and correct it.
