CIS-74 - Computer Software Quality Assurance

De Anza College

Spring 2008

Browsercam Assignment

(Formerly known as the "Field Trip Assignment")

  • Watch instructor demo of how to use browsercam.com for browser-compatibility testing.

  • Register for free 24-hour trial at browsercam.com.

  • Use the browsercam trial to do browser-compatibility testing of the area of De Anza's web site which you are covering in your TDS/TCS.

    It is probably a better use of your trial's 100 images to use several URLs with only the most major browser-OS platforms, as De Anza's web team is not going to be terribly interested in an obscure browser-OS not handling a page properly. If the area you are testing does not involve several URLs, feel free to also test other areas and/or some of the spreadelove.com pages.

  • File a bug for each browser-compatibility problem you find.

  • Update your TDS to include one or more test cases that you did using browsercam.com. Be sure to specify which browser-OS platforms should be selected by the tester.

  • To hand in this assignment, send the URL of your browsercam Project page to me at mamp@earthlink.net with a Subject line of BROWSERCAM ASSIGNMENT. Make sure your real name as it appears in college records is at the top of the message body.

  • Your hand-in assignment does not need to include all 100 images from your trial.
